About the selector:


Selectors main mission is to provide a basic, simple, and fun support to help you on your last mile decission making process
It doesn`t aim to remove your judgement from a decission making process, but it aims to offer an informed proposal based on your values.
The assumption is that you have already run through an initial process of filtering your option and now you are down to your last 3.
If you have more than 3 options, then probably you have too many ;)

I hope you enjoy it!

How to use the selector:


1. Write your criteria: ex. car body colour, engine power, etc.
2. Select from 1-5 how important the criteria is for you. You can select a value only for a criteria and once it is selected, you can`t select same values for the others.
3. Select on each OPTION how it scores on each criteria based on a 1 to 10 ranking, 1 being the worst, 10 being the best. Here you can choose the same value for multiple criteria and options.
4. All fields must be completed.
5. Click on CALCULATE and it will return the result to help you decide.
6. Still can`t make up your mind, use the RANDOM generator and take a chance.
